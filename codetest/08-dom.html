*** 용어 정리 'DOM 요소에 접근하기' getElementById() getElementsByClassName() getElementsByTagName() querySelector() querySelectorAll() getAttribute() setAttribute() ~DOM트리에 있는 노드들~ 요소(Element)노드: 모든 HTML 태그 텍스트(Text)노드: HTML 태그에서 사용하는 모든 텍스트 내용 속성(Attribute)노드: HTML 태그에 있는 모든 속성 주석(Comment)노드: 주석 createElement() 새 요소 노드를 만든다. createTextNode() 텍스트 내용이 있을 경우 텍스트 노드를 만든다. appendChild() 텍스트 노드를 요소 노드에 자식 노드로 추가한다. createAttribute() 속성 노드를 만든다. setAttributeNode() 속성 노드를 요소 노드에 연결한다. appendChild() 새로 만든 요소 노드를 부모 노드에 추가한다.
